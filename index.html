About Me

I major in computer graphic technology at Purdue University in 2019. When I was in school, I learned about web development including both front-end and back-end. Comparing with back-end web development, I enjoyed in front-end web development a lot more.

Purdue courses taught me skills like HTML and CSS, which is the basic skills one will need to know to create a simple website, but since I want to create a website that’s more interactive and engaging, I spent my own time looking up other resources, and become a self-taught front-end web developer.

For now, I am looking forward to join the team as a front-end web developer, and create user-friendly website for the users!
